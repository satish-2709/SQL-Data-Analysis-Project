# 📊 SQL Data Analysis & Visualization using Python

## 🔹 Overview
This project demonstrates a complete data analysis pipeline using Python and SQL. It processes sales data, performs SQL queries to extract insights, generates visualizations, and exports results into an Excel report.

---

## 🚀 Features
- Read and process CSV data using Pandas
- Store data in SQLite database
- Perform SQL queries (GROUP BY, ORDER BY, LIMIT)
- Identify top products and top-performing customers
- Generate bar chart visualization using Matplotlib
- Export analysis results to Excel

---

## 🛠️ Technologies Used
- Python
- Pandas
- SQLite
- Matplotlib
- Openpyxl
